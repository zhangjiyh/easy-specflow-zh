## 8. 依赖关系简述

- 上下游依赖：前端登录页提交凭证后，请求封装调用后端登录接口；后端签发 token；前端 store 保存 token，并在请求拦截器里注入头部。
- 配置 / 接口 / 数据流关系：`index.vue` -> `user.ts` -> `http.ts` -> `AuthController` -> `AuthService` -> `JwtTokenInterceptor`。

## 9. 风险点 / 不确定项

- 风险 1：后端错误码与前端拦截器处理不一致时，401 跳转逻辑会失真。
- 风险 2：refresh token 是否本轮纳入范围尚未完全明确。
