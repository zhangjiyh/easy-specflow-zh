# Vue + Spring 联调检查清单

## 1. 前端页面与交互

- [ ] 路由跳转与权限控制符合预期
- [ ] 页面首屏无报错
- [ ] 表单校验、空态、错误态完整
- [ ] 关键交互在桌面端与移动端表现正常

## 2. 接口契约

- [ ] 路径、方法、参数和返回结构一致
- [ ] 错误码和错误信息格式统一
- [ ] 分页、排序、过滤字段定义一致
- [ ] 日期与时区格式处理一致

## 3. 后端实现

- [ ] Controller 入参与返回值定义清晰
- [ ] Service 边界、事务和异常处理合理
- [ ] Repository / Mapper 查询性能可接受
- [ ] 日志具备必要的排查上下文

## 4. 工作流文档

- [ ] `任务说明.md` 已写清目标、范围和验收标准
- [ ] `执行计划.md` 已拆出执行步骤
- [ ] `执行计划.md` 不存在未勾选的已承诺步骤
- [ ] `进度记录.md` 已记录改动和验证摘要
- [ ] `验收记录.md` 已形成最终结论

## 5. 建议命令

前端：

```bash
pnpm lint
pnpm test
pnpm build
```

后端：

```bash
./mvnw test
./mvnw verify
```
